{# Stage breadcrumb — click any reachable stage to jump. A non-destructive navigation strip that lets users move freely between stages without going through Back / Next clicks. Backward jumps are always allowed (the previous stages still hold the user's data); forward jumps are gated by ``RuleDraft.can_advance_to_stage`` per stage and silently land at the highest reachable stage if the user clicks past their current readiness. Context expected from the route render: current_stage : int (0..4) — which stage the composer panel is showing reachable : list[bool] of length 5 — which stages are clickable Renders as a horizontal sequence with separators, mirroring the "1. Observation → 2. Detection → 3. Metadata → 4. Review → 5. Output" mental model. #}
